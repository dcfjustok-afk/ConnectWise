## 风险等级

- [ ] 🟢 **无风险**：纯新增，不影响已有功能
- [ ] 🟡 **低风险**：新增可选字段，前端可忽略
- [ ] 🟠 **中风险**：行为变更但前端已适配
- [ ] 🔴 **高风险**：破坏性变更，前端必须同步修改
