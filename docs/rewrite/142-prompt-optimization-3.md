# Step 142 Prompt 优化-3

## 8 字段执行记录
- AI角色：Spec
- AI工具：Skills + Copilot Chat
- MCP：FS
- 输入提示词：输出针对实时与流式场景的提示词优化模板（兼容/稳定/风险/Gate）
- AI Coding参与点：沉淀可复用 Prompt，减少后续步骤歧义与返工
- 执行命令：无
- 验收标准：至少 6 条可直接复用模板
- 失败最小修复：补足模板条目与验收字段

## 优化前后
- 优化前：请求偏功能实现，容易漏掉契约与门禁证据
- 优化后：请求默认包含契约验证、风险表、回归命令、Gate 判定

## Prompt 模板库（Phase D/E 可复用）
1. 兼容审查模板
- 输入：请审查本阶段对 RL 红线的达成情况，输出 PASS/BLOCK 与证据表
- 输出：红线对照表 + 阻断项 + 最小修复建议

2. 稳定性审查模板
- 输入：请审查重试/熔断/配额/超时行为，给出风险等级
- 输出：能力核查表 + 命令证据 + 风险处置

3. 风险收敛模板
- 输入：请将风险按 CLOSED/DEFERRED/ACCEPTED 分类，并执行最小补丁收敛
- 输出：风险清单 + 已执行补丁 + 回归结果

4. 契约测试模板
- 输入：为接口生成契约测试，覆盖 happy path + error path + edge case
- 输出：测试文件 + 关键断言 + 执行结果

5. Gate 模板
- 输入：按 Gate-1..Gate-5 给出是否可进入下一阶段
- 输出：Gate 明细 + 总判定 + 前置条件

6. 会话日志模板
- 输入：按 8 字段写入当前 Step 记录，附命令证据
- 输出：可审计日志条目
