# ConnectWise
一个在线思维导图协作平台

## OpenCode Commands

本仓库已提供 OpenCode 项目指令文件：`.opencode.md`，并已配置项目级自定义命令目录：`.opencode/commands/`。

在 OpenCode 会话中可直接使用以下命令语义：
- `/prepare-commit`
- `/conventional-commit`
- `/commit-check`

若命令未自动生效，请在会话开头补一句：
`请加载并遵守仓库根目录 .opencode.md 的规则。`

若仍未生效，请重启 OpenCode 会话（自定义命令通常在会话启动时加载）。
